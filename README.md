# cpu_usage

Prints cpu usage per virtual core. Use as a lightweight
system monitor for [GeekTool](http://projects.tynsoe.org/en/geektool/)
or [uebersicht](http://tracesof.net/uebersicht/) widgets.

```bash
$ make
$ ./cpu_usage
Core    Usage (%)
0       10.00
1       0.00
2       10.00
3       0.00
```
